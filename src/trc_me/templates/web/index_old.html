How it works

If you find something with a trc.me code on it, open a web browser, enter the code as the web address, and flag its location.

When you flag things, you can get trc.me points. Sometimes they are redeemable for real-world stuff like cellphone airtime, and shopping vouchers. Sometimes it's just fun!

Anyone can get codes to track where their things are.
